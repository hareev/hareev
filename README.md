## Featured Project — Vytal

**[Vytal](https://hareev.github.io/vytal/)** · AI-Powered CRM Intelligence Platform · Open Source · MIT Licensed · v0.3.0

An AI-powered headless intelligence layer that sits above your existing CRM systems — solving enterprise CRM sprawl (organizations running 3–12 concurrent CRM platforms) through automated diagnostics, unified pipeline management, and AI-driven remediation.

**Vytal Scan** — signature feature: automated CRM health diagnostic engine scoring five dimensions (Schema Hygiene, Automation Complexity, Data Quality, Security, User Adoption) on a 0–100 composite scale, with Claude API-generated remediation plans delivered in seconds.

Integrates directly with CRM platforms via native APIs (Dynamics 365 OData v4 live; Salesforce, HubSpot, Zoho in progress) — no middleware dependency, no vendor lock-in, no per-seat pricing.
